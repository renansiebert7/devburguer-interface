# 🍔 DevBurger

Meu primeiro projeto Full Stack desenvolvido durante minha jornada de estudos em desenvolvimento web.

Este projeto foi fundamental para meu aprendizado, pois foi a primeira vez que trabalhei com conceitos como autenticação JWT, controllers, rotas protegidas, migrations, integração entre múltiplos bancos de dados, pagamentos online e deploy em produção.

O objetivo foi construir uma aplicação completa de hamburgueria, permitindo que clientes realizem pedidos online e administradores gerenciem produtos, categorias e pedidos através de um painel administrativo.

## 🚀 Projeto Online

## ⚠️ Observação

Atualmente o projeto foi desenvolvido e otimizado apenas para uso em computadores (Desktop).

## ✨ Funcionalidades

### Cliente

* Cadastro de usuários
* Login com autenticação JWT
* Navegação por categorias
* Carrinho de compras
* Alteração de quantidades dos produtos
* Resumo do pedido
* Checkout integrado com Stripe
* Finalização de pedidos

### Administrador

* Painel administrativo
* Cadastro de produtos
* Edição de produtos
* Gerenciamento de categorias
* Controle de pedidos
* Alteração de status dos pedidos
